# HTML Container and Boxes

## About

This HTML program creates a simple webpage with a **container** that holds four boxes. Each box uses a common `box` class for shared styling and an additional class (`box1`, `box2`, `box3`, `box4`) for individual styles such as different background colors. The external `style.css` file is linked using the `<link>` tag to apply CSS styles. This example demonstrates the use of HTML structure, class attributes, and external CSS to build a simple webpage layout.
